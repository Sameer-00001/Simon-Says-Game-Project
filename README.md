# Simon-Says-Game-Project
This is a game to improve your brain memory.

## How to Play
1. Click the Any button on your keyboard to begin the game.
2. Watch the sequence of colors that light up.
3. Repeat the sequence by clicking the colors in the same order.
4. Each round adds one more color to the sequence.
5. If you make a mistake, the game ends and you can restart.
